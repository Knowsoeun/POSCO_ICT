### VS에서 .js 파일 생성 후 cmd에서 한꺼번에 실행하는 법 
1. .js 파일 저장 후 cmd 열기  
2. cd 폴더 경로 들어가기  
3. 파일 이름이 app.js라면, ```cmd node app.js```라고 치면 한번에 실행된다.
