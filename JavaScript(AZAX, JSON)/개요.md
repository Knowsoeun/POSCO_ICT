## 1. 개요  

### 1. 개요 
클라이언트 : 서버 = N : 1  
서버는 DB와 실시간으로 지속적인 통신을 통해 데이터를 주고받는다.

## 2. 언어 특징  
### 1. 통신 데이터  
- **JSON**  
```json
[
    {
        "title": "배꼽",
        "author": "오쏘 라지니쉬",
        "price": "8,100",
        "date": "2001/11/02"
    },
    {
        "title": "화폐전쟁",
        "author": "쑹훙빙",
        "price": "18,000",
        "date": "2010/03/23"
    }
]
    
```
- **XML**  
```xml
<?xml version="1.0" encoding="UTF-8"?>
<고객들>
	<고객>
		<번호>1</번호>
		<이름>홍길동</이름>
		<주소>서울시</주소>
		<방문>2022/03/12</방문>
	</고객>
	<고객>
		<번호>2</번호>
		<이름>성춘향</이름>
		<주소>남원시</주소>
		<방문>2021/12/24</방문>
	</고객>
</고객들>
```
